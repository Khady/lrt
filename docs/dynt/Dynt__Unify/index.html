Unification

module type PARAM = sig ... end
val init : modulo_props:bool -> (module PARAM)

The unification algorithm can be parametrized. Currently, the only parameter is modulo_props. It allows the user to specify whether properties are ignored or interpreted as distinguishing feature of the types.

exception Not_unifiable

Is raised by the following functors whenever unification is not possible.
