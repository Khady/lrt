Initialization and transformation

val of_ttype : 'a Dynt.Ttype.t -> 'a t

Produce a representation of a dynamic type, that is safer to inspect than the Stype.t one.

val xtype_of_ttype : 'a Dynt.Ttype.t -> 'a xtype

See of_ttype. This produces the xtype without wrapping it in t.

val remove_outer_props : 'a t -> 'a t

Strip the outer properties from the type. The result is guaranteed to be different from Prop.

val consume_outer_props : 'a t -> Dynt.Stype.properties * 'a t

See remove_outer_props. This function additionally returns an accumulated list of the removed properties. The accumulation is associative, i.e. Prop ([a], Prop ([b, c], _)) yields the same list as Prop ([a, b], Prop ([c], _)) .

Returns all the paths leading to a value of type 'a inside a value of type 'b. Does not traverse list, array, lazy and objects. Will loop on recursive types.
