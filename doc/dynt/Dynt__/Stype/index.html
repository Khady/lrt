val consume_outer_props : t -> properties * t

Read the properties from a stype and returns a stype that is not a property node. If properties are nested, the innermost properties are at the beginning of the return list.

In the sense of this function the following types s and t carry the same list of properties.

type s' = int [@prop {a = "b"; b = "c"}] [@@deriving t]
type s = s' [@prop {c = "d"; d = "e"}] [@@deriving t]
type t = int [@prop {a = "b"; b = "c"; c = "d"; d = "e"}] [@@deriving t]
